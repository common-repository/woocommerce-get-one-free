A WooCommerce extension which allows for giving the customer a free item when purchasing a specified quantity of this item.

== Description ==

**Note:** This extension is no longer under development.

WooCommerce Get One Free is a WooCommerce extension which allows for creating special offers for customers to get one item for free when purchasing certain quantity of this item.

WooCommerce 2.0.x is required.

== Installation ==

1. Download the latest version of the extension and extract it in the '/wp-content/plugins/' directory. 
2. Activate the plugin through the 'Plugins' menu in WordPress. 

For further information about installing a WordPress plugin, refer to http://codex.wordpress.org/Managing_Plugins

**Usage**

*General Settings*

After installing the extension, optionally navigate to WooCommerce > Settings and click on the 'Get One Free' tab. On this tab you can find general options for this extension.

*   You can disable the message on the cart page motivating the customer to purchase more items in order to get the free item. This message is enabled by default. 
*   You can redefine the default message from the previous point.. You can use the placeholders %PRODUCT% and %QUANTITY% which will be automatically replaced by the actual product with Get One Free feature set and minimal quantity for which this Special Offer applies, respectively. 
*   In a similar way you can set the message for the case when the customer is actually eligible for the Special Offer. 
*   You can also redefine the CSS style for the 'Free' label which is displayed next to the extra free item in the order. 

*Product Settings*

You can apply the Get One Free Offer for any WooCommerce product. Just navigate to Products and choose to edit the desired product. Click on the Get One Free tab and enable the feature for this product. Enter the minimal quantity for which the Special Offer applies and enter the message which will be displayed on the frontend Product page.
